# Smart Trees - 智慧树

## 项目简介

Smart Trees 是一个创新的 **3D 知识可视化与管理系统**，将复杂的知识结构转化为交互式的三维森林。通过 AI 辅助生成和直观的可视化界面，帮助用户轻松构建、探索和管理知识体系。

## 核心亮点

### 🌲 3D 知识森林
- 使用 Three.js 构建的沉浸式 3D 知识网络
- 拖拽旋转、缩放浏览，点击节点查看详情
- 将抽象知识转化为直观的视觉体验

### 🤖 AI 智能助手
- 集成字节火山方舟 API，自动生成知识分支
- 森林向导聊天界面，自然语言操控知识树
- 智能内容扩展，一键丰富知识体系

### 📺 多媒体资源整合
- **YouTube 视频自动检索**：根据知识点语义匹配相关视频
- 一键获取与知识节点相关的学习资源

### ⚡ 即开即用
- 纯前端实现，无需构建工具
- CDN 动态加载，本地一键部署
- 响应式设计，支持多端访问

## 技术栈

| 类别 | 技术 |
|------|------|
| 前端框架 | React 18 + Hooks |
| 3D 渲染 | Three.js |
| 样式 | Tailwind CSS |
| AI 接口 | 字节火山方舟 API |
| 转译 | Babel Standalone |

## 快速开始

```bash
# 克隆项目
git clone <repository-url>
cd smart-trees

# 启动本地服务器
python -m http.server 8000

# 访问 http://localhost:8000
```
