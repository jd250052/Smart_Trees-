# Smart Trees - 知识森林

Smart Trees 是一个基于 Web 的知识可视化与管理系统，利用 3D 技术和 AI 能力，帮助用户构建、浏览和管理复杂的知识体系。

## 功能特性

### 🎯 核心功能
- **3D 知识树可视化**：通过 Three.js 实现的交互式 3D 知识网络，直观展示知识结构
- **AI 辅助内容生成**：集成字节火山方舟 API，自动生成知识分支和内容
- **多视图切换**：支持 2D 树状视图、3D 网络视图和目录视图
- **知识节点管理**：创建、编辑、删除和组织知识节点
- **交互式聊天助手**：通过森林向导聊天界面操控知识森林
- **YouTube 视频检索**：根据节点语义自动匹配相关 YouTube 视频

### 📱 用户体验
- **响应式设计**：适配不同屏幕尺寸
- **流畅动画**：精心设计的过渡和交互动画
- **直观操作**：拖拽旋转 3D 视图，点击查看节点详情
- **实时反馈**：操作即时响应，提供视觉反馈

### 🔧 技术特性
- **无构建部署**：使用 CDN 链接和动态转译，无需构建系统
- **API 配置**：内置 API Key 管理界面，支持自定义 AI 模型
- **本地存储**：API Key 和模型配置自动保存到本地存储
- **模块化设计**：清晰的代码结构，便于维护和扩展

## 技术栈

- **前端框架**：React 18
- **3D 渲染**：Three.js
- **样式**：Tailwind CSS
- **AI 集成**：字节火山方舟 API（OpenAI 兼容）
- **数据存储**：Firebase（可选，用于持久化存储）
- **转译**：Babel（动态转译 JSX）

## 快速开始

### 1. 克隆项目

```bash
git clone <repository-url>
cd smart-trees
```

### 2. 启动本地服务器

```bash
# 使用 Python 内置服务器
python -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000
```

### 3. 访问应用

在浏览器中打开：http://localhost:8000

### 4. 配置 AI API

1. 在应用侧边栏中点击"设置"按钮
2. 输入您的字节火山方舟 API Key
3. 选择您想要使用的模型（默认：doubao-seed-1-6-251015）
4. 点击"保存"按钮

## 使用指南

### 创建知识树

1. 在首页输入知识主题
2. 点击"生长"按钮
3. AI 将自动生成相关的知识分支
4. 您可以继续编辑和扩展知识树

### 浏览 3D 视图

1. 创建或加载知识树后，点击顶部导航栏的"3D"按钮
2. 在 3D 视图中：
   - **拖拽**：旋转整个知识网络
   - **滚轮**：缩放视图
   - **点击节点**：查看节点详细信息

### 使用森林向导

1. 点击右侧侧边栏的"森林向导"图标
2. 在聊天输入框中输入您的请求，例如：
   - "创建一个关于机器学习的知识树"
   - "切换到 3D 视图"
   - "展开所有节点"
3. 森林向导将执行您的命令并提供相应的反馈

### 使用 YouTube 视频检索

1. 在 2D 视图中，找到并点击一个知识节点
2. 在节点详情页面中，点击"检索资源"按钮
3. 系统将自动根据知识点关键词匹配相关的 YouTube 视频
4. 匹配的视频将显示在节点下方，点击视频卡片可以查看详细信息
5. 视频检索结果基于关键词匹配，会优先显示与知识点最相关的视频

## 配置说明

### API Key 配置

应用使用字节火山方舟 API 进行 AI 内容生成。您需要在侧边栏的设置界面中配置您的 API Key。

### 模型选择

默认使用 `doubao-seed-1-6-251015` 模型，您可以根据需要选择其他兼容的模型。

### Firebase 配置

当前使用的是演示配置，用于展示目的。如果您需要持久化存储功能，可以修改 `index.html` 中的 Firebase 配置：

```javascript
window.__firebase_config = JSON.stringify({
    apiKey: "your-api-key",
    authDomain: "your-project.firebaseapp.com",
    projectId: "your-project-id",
    // 其他配置...
});
```

## 常见问题

### Q: 为什么会出现 "请求失败: 请先配置 Ark API Key" 的提示？

**A:** 这是因为您尚未配置字节火山方舟 API Key。请在侧边栏的设置界面中输入您的 API Key。

### Q: 3D 视图加载缓慢怎么办？

**A:** 3D 视图需要加载 Three.js 库，首次加载可能会稍慢。后续加载会使用浏览器缓存，速度会加快。

### Q: 应用可以离线使用吗？

**A:** 是的，基本功能可以离线使用，但 AI 内容生成需要网络连接。
